## 安装pip3
```bash
sudo apt-get install python3-pip
```

## 安装Django
```bash
sudo pip3 install django
```

![2019-03-26T06:49:38.png][1]

## 创建项目
```bash
django-admin startproject convert
```

## 运行
```bash
python3 manage.py runserver
```
