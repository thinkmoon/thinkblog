## rpx单位
> rpx（responsive pixel）: 可以根据屏幕宽度进行自适应。规定屏幕宽为750rpx。要实现4:3的展示则可以
```css
image{
    width:750rpx;
    height:563rpx;
}
```
## 使用组件库，写的css样式优先级太低
> 不管三七二十一，加`!important`
```css
.findButton {
  width:300px !important;
  height:35px !important;
}
```
## 样式各种飘，不居中怎么办?
```html
<view class="nickName">
  <view>{{userInfo.nickName}}</view>
</view>
```
```css
.nickName {
  width: 100%;
  display: flex;
  align-items: center;
  justify-content: center;
  height: 44px;
}
```
## selectedColor无效
> selectedColor  和 list 是同级别的。写在list数组外面就可以了。

## 背景虚化
> blur中的参数是虚化比例
```css
filter: blur(1px);
```
## `scroll-view`设置`scroll-x"也不会横向滚动
要给scroll-view加上white-space: nowrap; ，给scroll-view的子元素box加上display:inline-block;
