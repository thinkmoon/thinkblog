nodejs的一些笔记

2019-03-12 学习笔记
标签: nodejs

安装nodejs


sudo apt-get install nodejs

sudo apt-get install npm

全局安装包


sudo npm i -g cnpm

查看全局安装的包


npm list -g --depth 0

全局卸载包


sudo npm uninstall -g cnpm

清理缓存


npm cache clean -f

默认镜像太慢换淘宝镜像


npm config set registry https://registry.npm.taobao.org

现在我常使用cnpm包

检查配置是否成功


npm config get registry
