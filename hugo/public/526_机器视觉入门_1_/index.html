遇到的一些问题

Imageio: ‘ffmpeg-linux64-v3.3.1’ was not found on your computer; downloading it now.

解决办法：

sudo add-apt-repository -y ppa:djcj/hybrid && sudo apt update && sudo apt install -y ffmpeg

我直接运行了，然后它自动下载安装了。
